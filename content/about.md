近期我一直在公司负责数据库和 AI 交互的相关工作，期间搭建了许多 MCP 和 Agent 系统。突然萌生了一个想法：为什么不把个人博客作为一个实际应用场景呢？

于是，这个博客便诞生了。

目前，您会在页面右下角看到一个按钮，点击它会打开一个侧边的聊天 Agent。这个 Agent 不仅可以将我的博客内容作为知识库，还会记录和存储读者的反馈，形成一个动态的记忆系统。

与它对话，就像直接与我对话一样。

**实现原理**：MCP 工具注册在公司的数据库中，Agent 部署在 Dify 云服务器上，前端界面则是通过 Hugo 和 Cloudflare 搭建的。
